vim一些設定參數

ubuntu預設的vim Normal 和 Insert Mode 鼠標無法分辨之解決方法：

youtube解說

字體設定

ubuntu: 

set guifont:Monospace\ 15
windows: 

set guifont:Monospace:h15

自動loading


set autoread

Jump back previous position

Control + o

Jump next position

Control + i

auto loading, tab width

"********************read file auto
set autoread
"************ stop tab
set noai
"************ tab width
set tabstop=4
set shiftwidth=4
"************ tab convert to space
set expandtab

Setting of text font on gvim

" Windows set guifont=<font>:h<size> " ubuntu set guifont=<font\ font>\ <size>
