整數類型的轉換

為了防止整數溢位，可以在其數值的後方加上一個L, 強制轉換成long類型。 
int i=java.lang.Integer.MAX_VALUE; //讓java提供整數的最大值
System.out.println("Value: "+(i+100L)); // 會使i的結果為long 類型
