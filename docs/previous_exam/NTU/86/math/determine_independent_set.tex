題目\\

解:\\
題目說$\{Av_1,Av_2,\dots,Av_k\}$為線性獨立集，\\
則純在k個純量都為0視為唯一解\\
$\Rightarrow \alpha_1Av_1+\alpha_2Av_2+\dots+\alpha_kAv_k=0, \alpha_1=\alpha_2=\dots=\alpha_k=0$\\
$\Rightarrow (\alpha_1A)v_1+(\alpha_2A)v_2+\dots+(\alpha_kA)v_k=0$\\
$\Rightarrow 0\cdot v_1+0\cdot v_2+\dots+0\cdot v_k=0$\\
所以$\{v_1,v_2,\dots, v_k\}$為線性獨立集。
