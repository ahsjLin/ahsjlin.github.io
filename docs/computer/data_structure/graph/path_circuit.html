Path

滿足條件, 原點不是終點

Circuit(Cycle)

滿足條件, 原點等於終點

Euler

cycle(circuit): 每條路都走過一遍, 原點等於終點。
path: 每條路都走過一遍，原點不等於終點。

如何判斷是不是Euler 迴路/路徑

每個點都有他的degree(分支度)， 每個點需要是偶數的分支度，才能夠形成euler 迴路/路徑。
